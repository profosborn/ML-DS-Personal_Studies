# 📁 Data Science Projects

Store your completed DS projects here. Each project should be its own folder with a proper README.

## Project Template Structure

```
project-name/
├── README.md              ← Business problem, methodology, findings
├── notebooks/
│   ├── 01_eda.ipynb       ← Exploratory Data Analysis
│   ├── 02_modeling.ipynb  ← Model building
│   └── 03_analysis.ipynb  ← Final analysis
├── src/
│   ├── data_loader.py     ← Data loading utilities
│   ├── features.py        ← Feature engineering
│   └── visualize.py       ← Visualization functions
├── data/
│   └── raw/               ← Raw data (or instructions to download)
├── reports/
│   └── findings.pdf       ← Final report/presentation
├── requirements.txt
└── .gitignore
```
