# 📊 Ghana Economic Dashboard

## 🎯 Business Problem
_Analyze Ghana's key economic indicators to identify trends, patterns, and insights that inform understanding of the country's economic health._

## 📋 Status: 🔲 Not Started

## 🛠️ Tech Stack
- **Data Collection:** Web scraping, APIs (Bank of Ghana, World Bank)
- **Database:** PostgreSQL
- **Analysis:** SQL, Python (Pandas)
- **Visualization:** Streamlit / Plotly
- **Deployment:** Streamlit Cloud

## 📊 Key Questions
1. What is the GDP growth trend over the past decade?
2. How has inflation affected the GHS/USD exchange rate?
3. Which sectors contribute most to Ghana's GDP?
4. What is the correlation between interest rates and inflation?
5. How did COVID-19 impact key economic indicators?

## 🔬 Skills Demonstrated
- SQL (complex queries, window functions, CTEs)
- Data visualization and storytelling
- Dashboard deployment
- Domain knowledge (economics, Ghana)
